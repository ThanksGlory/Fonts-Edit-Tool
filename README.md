# Fonts-Edit-Tool
### 离线字体合并，编辑软件

该软件适合在完全内网开发的小伙伴，对字体进行编辑、合并、导出、发版等需求。

- [x] 支持png导入字体；
- [x] 支持svg导入字体；
- [x] 支持ttf字体识别导入；
- [x] 支持ttf拖拽导入；
- [x] 支持导出单个和批量导出svg；
- [x] 支持对某一字体图形的放大、缩小、平移、旋转，以及图形的svg替换；
- [x] 支持导出iconfont字体下载包；
- [x] 支持单独导出ttf字体文件；
- [ ] 添加名称刷新排序；（未完成）
- [ ] 添加字体配置功能，例如：修改字体名字、修改icon前缀等；（未完成）
- [ ] 暂不支持彩色图标；（未完成）
